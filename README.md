# Scented Lads - Official Website

Premium scented candles and home fragrance manufacturer.

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Language:** TypeScript

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```
